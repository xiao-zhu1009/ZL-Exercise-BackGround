# 【任务书】训练模块

---

## 1. 需求说明

为用户端新增完整的训练中心模块，包含三大功能：
- **训练记录**：用户手动记录每次训练（类型、时长、消耗卡路里、备注），支持增删查
- **训练计划**：查看教练为自己制定的近期训练计划（进行中/已完成/已终止）
- **数据可视化**：图表展示近 7/30 天训练趋势（训练时长柱状图、卡路里折线图、训练类型分布饼图）

---

## 2. 实现思路

- 复用已有 `WorkoutRecord`、`TrainingPlan`、`CoachStudent` 模型（models/training.py 已定义）
- 后端新增 `api/training.py` 路由 + `CRUD/training.py` 数据层
- 前端新增 `views/user/Training/` 目录，包含主页面（Tab 切换三个子模块）
- 图表使用 ECharts（CDN 引入或 npm 安装 echarts）

---

## 3. 前端要做什么

| 文件 | 职责 |
|------|------|
| `views/user/Training/index.vue` | 训练中心主页，Tab 切换三个子模块 |
| `views/user/Training/WorkoutLog.vue` | 训练记录子组件：列表 + 新增弹窗 |
| `views/user/Training/TrainingPlan.vue` | 训练计划子组件：卡片展示近期计划 |
| `views/user/Training/TrainingChart.vue` | 数据可视化子组件：ECharts 图表 |
| `api/training.js` | 封装所有训练相关 axios 请求 |

路由新增：`/user/training` → `Training/index.vue`

---

## 4. 后端要做什么

| 文件 | 职责 |
|------|------|
| `api/training.py` | 训练记录 + 训练计划 + 统计图表接口 |
| `CRUD/training.py` | 数据库异步操作层 |
| `schemas/training.py` | Pydantic 请求/响应 Schema |
| `main.py` | 注册新路由 |

---

## 5. 接口设计

### 训练记录
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/training/records` | 查询训练记录列表（分页，可按日期筛选） |
| POST | `/training/records` | 新增一条训练记录 |
| DELETE | `/training/records/{id}` | 删除一条训练记录 |

### 训练计划
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/training/plans` | 查询当前用户的训练计划列表（status 可选筛选） |

### 数据统计（图表）
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/training/stats?days=7` | 近 N 天每日训练时长+卡路里汇总（折线/柱状图） |
| GET | `/training/stats/type?days=30` | 近 N 天训练类型分布（饼图） |

---

## 6. 数据表结构

复用 `models/training.py` 中已有的三张表，无需新建：

### workout_records（训练记录）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int PK | 自增主键 |
| user_id | int | 用户 ID |
| record_date | date | 训练日期 |
| duration | int | 训练时长（分钟） |
| calories | int | 消耗卡路里 |
| workout_type | varchar(50) | 训练类型（力量/有氧/瑜伽/球类/其他） |
| note | varchar(500) | 备注 |
| extra | json | 扩展字段（动作组数等） |

### training_plans（训练计划）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int PK | 自增主键 |
| coach_id | int | 制定计划的教练 ID |
| student_id | int | 学员 ID（即当前用户） |
| title | varchar(200) | 计划标题 |
| goal | varchar(50) | 目标（减脂/增肌/塑形/提升耐力） |
| start_date | date | 开始日期 |
| end_date | date | 结束日期 |
| description | text | 计划描述 |
| content | json | 每日训练内容（结构化） |
| status | smallint | 0=已终止 1=进行中 2=已完成 |
