# 【任务书】训练记录编辑 + 动作库搜索提示

## 1. 需求说明
- 训练记录支持编辑（修改已有记录的所有字段，包括详细动作）
- 详细模式录入动作时，可搜索动作库快速填入动作名称

## 2. 实现思路
- 编辑：复用新增弹窗，区分 create/edit 模式，编辑时回填数据，调用 PUT 接口
- 动作库搜索：动作名称输入框改用 el-autocomplete，输入时调 GET /actions?keyword=xxx 拿建议列表

## 3. 前端改动
- WorkoutLog.vue：列表加编辑按钮，弹窗支持 edit 模式回填，动作名称改为 autocomplete
- api/training.js：加 updateWorkoutRecord 方法

## 4. 后端改动
- api/training.py：加 PUT /training/records/{id} 接口
- CRUD/training.py：加 update_workout_record 函数

## 5. 接口设计
PUT /training/records/{id}
body 同 POST，返回更新后的记录

## 6. 数据表结构
无需变更
