# 【任务书】训练记录详细模式

## 1. 需求说明
训练记录支持两种录入模式：
- 简单模式：类型 + 时长 + 卡路里 + 备注（原有）
- 详细模式：在简单模式基础上，可逐条录入具体动作（如"杠铃卧推 4组×12次 80kg"）

## 2. 实现思路
`workout_records` 表已有 `extra` JSON 字段，直接用于存储详细动作列表，无需改表。

extra 结构：
```json
{
  "mode": "detail",
  "exercises": [
    { "name": "杠铃卧推", "sets": 4, "reps": 12, "weight": 80, "unit": "kg", "note": "" }
  ]
}
```
简单模式 extra 为 null。

## 3. 前端改动
- WorkoutLog.vue：新增弹窗加模式切换（简单/详细）
- 详细模式下显示动作列表编辑区，支持增删条目
- 列表行支持展开查看动作明细

## 4. 后端改动
- schemas/training.py：WorkoutRecordCreate 加 extra 字段
- api/training.py：list_records 和 add_record 返回带 extra

## 5. 接口设计
POST /training/records
新增 extra 字段（可选），简单模式不传或传 null

GET /training/records 返回每条记录带 extra 字段

## 6. 数据表结构
无需变更，extra JSON 字段已存在
