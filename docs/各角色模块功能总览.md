# 各角色模块功能总览

---

## 一、用户端（role = user）

### 1. 账号与个人中心
- 注册 / 登录
- 个人主页查看与编辑（头像、昵称、基本信息）
- 申请成为教练（填写申请说明、查看审核状态）

### 2. 训练中心
- **训练记录**：新增 / 查看 / 删除训练记录（类型、时长、卡路里、备注）
- **训练计划**：查看教练为自己指派的计划（进行中 / 已完成 / 已终止）
- **数据统计**：近 7/30 天训练时长柱状图、卡路里折线图、训练类型饼图

### 3. 饮食中心
- **饮食记录**：记录每日饮食
- **饮食文章**：浏览文章列表（分类筛选 + 关键词搜索）、查看文章详情
- **食物库**：搜索食物（仅显示审核通过的）

### 4. 课程中心
- **课程列表**：浏览已上线课程（分类筛选 + 关键词搜索 + 分页）
- **课程详情**：查看课程完整信息
- **预约管理**：申请预约、查看我的预约列表、取消预约

### 5. 教练专区
- **教练列表 / 搜索**：浏览可绑定的教练
- **教练详情**：查看教练简介、资质
- **绑定管理**：发起绑定申请、查看申请状态、查看当前绑定教练

---

## 二、教练端（role = coach）

### 1. 账号
- 登录（由管理员审批通过后 role 变为 coach，无需重新注册）

### 2. 学员管理
- **学员列表**：查看绑定中学员 / 待处理申请（Tab 切换）
- **学员详情**：基本信息、最近训练记录摘要
- **申请处理**：同意 / 拒绝学员绑定申请

### 3. 训练计划管理
- 为学员创建训练计划
- 查看 / 管理某学员的计划列表

### 4. 课程管理
- **发布课程**：填写课程信息 + 封面上传，提交后进入待审核
- **我的课程列表**：查看各状态课程（待审核 / 招募中 / 满员 / 已结束 / 已驳回 / 已下架）
- **修改课程**：仅限已驳回状态，修改后重新提交审核
- **删除课程**：仅限待审核 / 已驳回状态
- **预约审批**：查看课程预约申请列表、同意 / 拒绝预约

### 5. 饮食文章管理
- **发布文章**：封面上传 + 富文本内容，提交后进入待审核
- **我的投稿列表**：查看各状态文章（待审核 / 已上线 / 已驳回）
- **修改文章**：仅限已驳回状态，修改后重新提交审核
- **删除文章**：仅限待审核 / 已驳回状态

### 6. 食物库投稿
- 投稿新食物
- 查看自己的投稿列表（待审核 / 已通过 / 已驳回）
- 删除待审核 / 已驳回的投稿

---

## 三、管理员端（role = admin）

### 1. 教练申请审核
- 申请列表（Tab：待审 / 已通过 / 已拒绝）
- 通过申请（用户 role 变为 coach，立即生效）
- 拒绝申请（需填写拒绝原因）

### 2. 课程审核
- 课程列表（按状态筛选：待审核 / 招募中 / 满员 / 已结束 / 已驳回 / 已下架）
- 审核课程：通过（status=1 招募中）/ 驳回（需填原因）
- 下架已上线课程（招募中 / 满员 → 已下架）

### 3. 饮食文章审核
- 文章列表（按状态筛选：待审核 / 已上线 / 已驳回 / 已下架）
- 审核文章：通过 / 驳回（需填原因）
- 下架已上线文章

### 4. 食物库审核
- 投稿列表（按状态筛选：待审核 / 已通过 / 已驳回）
- 通过 / 驳回食物投稿

### 5. 教练-学员关系监管（P1 规划）
- 绑定关系总览：绑定数、待处理申请数、解绑率
- 教练维度统计：名下学员数、下发计划数
- 学员维度统计：绑定教练数、计划完成度
- 操作审计日志：关键操作记录（建计划、解绑等）
