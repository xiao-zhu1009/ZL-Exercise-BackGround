# 【任务书】训练中心-身体记录模块

## 1. 需求说明
在训练中心新增"身体记录"标签页，支持按日期记录体重/身高/BMI/体脂率，可查看历史趋势，支持新增、编辑、删除。

## 2. 实现思路
body_records 表已存在（user_id、record_date、height、weight、bmi、body_fat、remark），直接补接口。
BMI 后端自动计算（weight / (height/100)²）。
前端在训练中心 tabs 加新 tab，独立组件展示列表 + 体重折线图 + 新增/编辑弹窗。

## 3. 前端改动
- 新建 Training/BodyRecord.vue
- Training/index.vue 加 tab 入口
- api/training.js 加 body record 相关方法

## 4. 后端改动
- CRUD/training.py 加 body record 增删改查函数
- api/training.py 加 4 个接口

## 5. 接口设计
GET    /training/body-records?days=90     查询近N天记录（按日期升序，用于图表）
POST   /training/body-records             新增
PUT    /training/body-records/{id}        修改
DELETE /training/body-records/{id}        删除

## 6. 数据表结构
无需变更，body_records 已存在
字段：user_id, record_date, height, weight, bmi(自动算), body_fat, remark
