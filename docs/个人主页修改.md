# 个人主页修改 技术文档

## 【任务书】

1. **需求说明**：个人主页数据目前全部写死，需改为从数据库读取真实数据，支持查看和修改账号信息、身体指标、修改密码
2. **实现思路**：users 表加 `signature` 字段；新建 `user_body_stats` 表存身体指标；后端提供 5 个接口；前端 created 时调接口填充数据，保存时调修改接口
3. **前端要做什么**：`Profile/index.vue` 的 created 调 GET 接口拉数据，三个保存方法改为真实 axios 请求
4. **后端要做什么**：新增 `api/user.py`，注册到 `main.py`，提供 5 个接口
5. **接口设计**：见下方
6. **数据表结构**：见下方

---

## 接口设计

### 1. 获取个人信息
- **Method**：`GET`
- **Path**：`/user/profile`
- **Auth**：需要 Token（Bearer）
- **Response**：
```json
{
  "code": 200,
  "msg": "ok",
  "data": {
    "username": "user001",
    "nickname": "健身达人",
    "phone": "138****8888",
    "signature": "每天进步一点点",
    "avatar": ""
  }
}
```

---

### 2. 修改个人信息
- **Method**：`PUT`
- **Path**：`/user/profile`
- **Auth**：需要 Token
- **Request Body**：
```json
{
  "nickname": "新昵称",
  "phone": "13900009999",
  "signature": "新签名"
}
```
- **Response**：`{ "code": 200, "msg": "保存成功", "data": null }`

---

### 3. 获取身体指标
- **Method**：`GET`
- **Path**：`/user/body-stats`
- **Auth**：需要 Token
- **Response**：
```json
{
  "code": 200,
  "msg": "ok",
  "data": {
    "height": 175,
    "weight": 70.5,
    "bmi": "23.0",
    "body_fat": 18.5,
    "waist": 80,
    "hip": 95,
    "whr": "0.84"
  }
}
```
> 若用户尚未填写，data 中各字段返回 null

---

### 4. 修改身体指标
- **Method**：`PUT`
- **Path**：`/user/body-stats`
- **Auth**：需要 Token
- **Request Body**：
```json
{
  "height": 175,
  "weight": 70.5,
  "body_fat": 18.5,
  "waist": 80,
  "hip": 95
}
```
> bmi 和 whr 由后端根据 height/weight/waist/hip 自动计算，不需要前端传
- **Response**：`{ "code": 200, "msg": "保存成功", "data": null }`

---

### 5. 修改密码
- **Method**：`PUT`
- **Path**：`/user/password`
- **Auth**：需要 Token
- **Request Body**：
```json
{
  "old_password": "旧密码",
  "new_password": "新密码"
}
```
- **Response**：`{ "code": 200, "msg": "密码修改成功", "data": null }`
- **Error**：`{ "code": 400, "msg": "旧密码错误", "data": null }`

## 数据表结构

### users 表新增字段
```sql
ALTER TABLE users ADD COLUMN signature VARCHAR(100) NOT NULL DEFAULT '';
```

### 新建 user_body_stats 表
```sql
CREATE TABLE user_body_stats (
  id          INT PRIMARY KEY AUTO_INCREMENT,
  user_id     INT NOT NULL UNIQUE,
  height      FLOAT,
  weight      FLOAT,
  bmi         FLOAT,
  body_fat    FLOAT,
  waist       FLOAT,
  hip         FLOAT,
  whr         FLOAT,
  created_at  DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
  updated_at  DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  is_deleted  TINYINT NOT NULL DEFAULT 0
);
```

## 前端需修改的文件

### `src/views/user/Profile/index.vue`

**data 部分**：profile 和 body 初始值改为空/null，等 created 拉取后填充

**新增 created**：
```js
async created() {
  // 拉账号信息
  const r1 = await this.$http.get('/user/profile')
  this.profile = r1.data.data

  // 拉身体指标
  const r2 = await this.$http.get('/user/body-stats')
  if (r2.data.data) {
    Object.assign(this.body, r2.data.data)
  }
}
```

**saveProfile 改为真实请求**：
```js
async saveProfile() {
  const { nickname, phone, signature } = this.profile
  await this.$http.put('/user/profile', { nickname, phone, signature })
  this.$message.success('账号信息已保存')
}
```

**saveBody 改为真实请求**：
```js
async saveBody() {
  const { height, weight, body_fat, waist, hip } = this.body
  const res = await this.$http.put('/user/body-stats', { height, weight, body_fat, waist, hip })
  // 后端返回计算好的 bmi 和 whr
  Object.assign(this.body, res.data.data)
  this.$message.success('身体指标已保存')
}
```

**changePassword 改为真实请求**：
```js
async changePassword() {
  const { old_password, new_password, confirm_password } = this.pwdForm
  if (new_password !== confirm_password) return this.$message.error('两次密码不一致')
  await this.$http.put('/user/password', { old_password, new_password })
  this.$message.success('密码修改成功')
  this.pwdDialog = false
  this.pwdForm = { old_password: '', new_password: '', confirm_password: '' }
}
```
